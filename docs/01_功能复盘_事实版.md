## 1）插件定位（当前实现 + 目标闭环）

## 3）用户可触发入口（现有）

### 3.1 命令（Command Palette）
- 性能报告：`think-performance-report`
- 重建索引：`think-rebuild-index`（清缓存 + 全量 rescan）
- 迁移工具：将 heading 主题写入任务行（active file / all files）
- AI Chat：`think-open-ai-chat`
- Timer：按 taskId 开始/继续；停止并写回

### 3.2 代码块渲染（Dashboard 嵌入笔记）
- 语言：`think`
- 支持：
  - `LayoutName`（纯文本）
  - `{{"layout":"LayoutName"}}`（JSON）
- 行为：从 settings 查 layout → renderer 渲染到 codeblock 区域

### 3.3 thinktxt（预览模式增强）
- 将连续的 thinktxt callout 在预览模式中合并、KV 解析并渲染为只读表格（编辑模式保留原始 KV 块）
- **边界规则固定**：你已确认“有明确边界标记，都是 KV 块”
