# 字体大小统一映射指南

## 📊 当前字体大小使用统计

基于代码扫描，项目中使用了以下字体大小：
- **8px**: 1处 (图表细节)
- **9px**: 2处 (最小标签) 
- **10px**: 8处 (图表标签)
- **11px**: 6处 (小标签、说明)
- **12px**: 15处 (正文内容)
- **13px**: 8处 (按钮、标签)
- **14px**: 12处 (标题、重要内容)
- **15px**: 2处 (主标题)
- **16px**: 4处 (页面标题)
- **18px**: 2处 (重要按钮)
- **20px**: 2处 (主要操作按钮)

## 🎯 新的统一字体大小系统

### 基础字体大小类名映射

| 原字体大小 | 新工具类 | 用途说明 | 示例场景 |
|------------|----------|----------|----------|
| `font-size: 8px` | `.think-text-2xs` | 最小字体：图表细节、水印 | 图表最小标签 |
| `font-size: 9px` | `.think-text-chart-mini` | 图表最小标签 | 图表细节文字 |
| `font-size: 10px` | `.think-text-xs` 或 `.think-text-chart-label` | 超小字体：图表标签、辅助信息 | 图表坐标轴标签 |
| `font-size: 11px` | `.think-text-sm` 或 `.think-text-chart-value` | 小字体：标签、说明文字 | 表单说明、数值显示 |
| `font-size: 12px` | `.think-text-base` 或 `.think-text-button-sm` | 基础字体：正文内容 | 正文、小按钮 |
| `font-size: 13px` | `.think-text-md` 或 `.think-text-button-md` | 中等字体：按钮、表单标签 | 中等按钮、重要标签 |
| `font-size: 14px` | `.think-text-lg` 或 `.think-text-button-lg` | 大字体：标题、重要内容 | 卡片标题、大按钮 |
| `font-size: 15px` | `.think-text-xl` | 超大字体：主标题 | 模块标题 |
| `font-size: 16px` | `.think-text-2xl` 或 `.think-text-icon-md` | 特大字体：页面标题 | 页面主标题、中等图标 |
| `font-size: 18px` | `.think-text-3xl` 或 `.think-text-icon-lg` | 巨大字体：重要按钮、图标 | 重要操作按钮、大图标 |
| `font-size: 20px` | `.think-text-4xl` 或 `.think-text-icon-xl` | 最大字体：操作按钮、主要图标 | 主要操作按钮、超大图标 |

## 🔄 实际替换示例

### CSS 样式替换

```css
/* ❌ 原始写法 */
.old-class {
    font-size: 12px;
    font-weight: 600;
    color: var(--text-normal);
}

/* ✅ 新的工具类写法 */
.old-class {
    color: var(--text-normal);
}
/* HTML中使用: <div class="old-class think-text-base think-text-semibold"> */
```

### HTML/JSX 替换

```tsx
// ❌ 内联样式写法
<div style={{ fontSize: '13px', fontWeight: 500 }}>按钮文字</div>

// ✅ 工具类写法
<div className="think-text-md think-text-medium">按钮文字</div>

// ❌ 复杂内联样式
<span style={{ 
    fontSize: '11px', 
    color: 'var(--text-muted)', 
    textAlign: 'center' 
}}>
    图表标签
</span>

// ✅ 工具类写法
<span className="think-text-chart-value think-text-muted think-text-center">
    图表标签
</span>
```
