# QuickInput 主线验收清单

## 1. 任务读取

- [ ] 编辑任务时，正文包含连续空格的内容不能被截断。
- [ ] 示例：`长治学院  设计道旗定稿` 必须完整显示。
- [ ] 任务前缀 `- [x]`、图标、完成日期、元数据不能混入正文输入框。

## 2. 时间计算

- [ ] 默认值注入 `时间 + 时长` 后能自动得到 `结束`。
- [ ] 默认值注入 `时间 + 结束` 后能自动得到 `时长`。
- [ ] 默认值注入 `结束 + 时长` 后能自动得到 `时间`。
- [ ] 用户手动改过的时间字段不要被主题默认值无声覆盖。

## 3. 写回与迁移保存

- [ ] 目标路径不变时，原地替换原记录。
- [ ] 目标路径变化时，顶部提示显示 `原路径 → 新路径`。
- [ ] 保存目标路径变化的记录时，先写入新位置，再删除旧记录。
- [ ] 如果旧记录删除失败，新位置记录必须保留，并提示用户手动清理旧记录。
- [ ] 写回任务时保留 tag、完成日期、模板 ID、模板来源、自定义 `(key::value)` 元数据。

## 4. 主题三分法

- [ ] `工作/设计` 能拆成：rootTheme=`工作`、leafTheme=`设计`、themePath=`工作/设计`。
- [ ] 视图设置中能筛选 rootTheme / leafTheme / themePath。

## 5. UI 与误操作保护

- [ ] QuickInput 点击外部不关闭。
- [ ] 桌面端输入时不出现手机输入法避让的大空白。
- [ ] 移动端输入时仍然有底部避让。
