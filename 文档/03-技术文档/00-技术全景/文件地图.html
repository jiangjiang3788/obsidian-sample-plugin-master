根目录地图

路径职责备注
manifest.jsonObsidian 插件元信息id、name、version、main、styles。
package.json依赖和脚本dev、build、check、gate、test。
src/main.ts插件主入口onload/onunload、命令注册、设置加载。
src/styles/main.css主样式入口引入或组织插件运行样式。
scripts/质量门禁和自动化脚本架构边界、DI、feature、capability 等 gate。
test/测试unit、integration、e2e、mocks、vaults。
docs/旧文档资料准备弃用，不作为正式入口。
html/旧 HTML 资料准备弃用，不作为正式入口。
文档/新项目驾驶舱当前正式文档入口。

src 目录地图

路径职责改什么时看
src/app应用组装、服务管理、capability、usecase、runtime改启动流程、依赖组合、能力入口。
src/core核心业务、数据、AI、主题、类型、工具改业务规则、数据结构、算法。
src/features面向用户的功能模块改 QuickInput、AI Chat、Settings、Timer。
src/platformObsidian API 适配改文件读取、元数据、弹窗、事件、UI port。
src/shared共享 UI、hooks、工具、样式改跨功能复用组件或工具。
src/types补充类型声明改外部库类型或全局类型补丁。

高频文件

文件职责注意
src/main.ts插件生命周期和命令注册避免放复杂业务逻辑。
src/app/ServiceManager.ts服务启动和清理改服务生命周期时看。
src/app/bootstrap/buildRuntime.ts构建运行时依赖改 port/usecase 组合时看。
src/app/capabilities/createCapabilities.ts能力入口组合AI/Timer 等功能统一入口。
src/features/settings/SettingsTab.tsx设置页入口设置 UI 改动先看。
src/core/services/DataStore.ts数据索引和存储相关改扫描、缓存、记录数据时看。
