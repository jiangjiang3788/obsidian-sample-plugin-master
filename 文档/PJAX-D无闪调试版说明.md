# PJAX-D 无闪调试版

本版相对上一版做了两件事：

1. 移除右侧正文 `.main` 的 opacity/transform 淡入淡出动画。上一版的轻过渡可能会被感知为“闪一下”。
2. 在控制台增加中文调试信息，便于判断当前点击到底是 PJAX 局部切页，还是回退成了浏览器整页刷新。

## 怎么看日志

打开浏览器开发者工具 Console，然后点击左侧目录。

正常局部切页应该看到：

- `[文档切页调试] 已拦截点击，准备局部切页`
- `[文档切页调试] 开始局部切页`
- `[文档切页调试] 开始请求目标 HTML`
- `[文档切页调试] 开始替换右侧正文 .main，左侧目录不会重建`
- `[文档切页调试] 局部切页完成`

如果每次点击都会出现新的 `页面生命周期 pageshow`，或者出现 `局部切页失败，将回退为浏览器普通跳转`，说明发生了整页刷新。

## 关闭日志

稳定后打开 `_资源/文档脚本.js`，把：

```js
const PJAX_DEBUG = true;
```

改成：

```js
const PJAX_DEBUG = false;
```
